# 报价与交易

报价与交易能力主要由插件承载，面向客户条款、产品利率报价、交易日报、North Info 和汇总导出等场景。

## 报价语义

- **客户报价条款**：客户 commission、financing、费率、返佣等字段，应使用客户管理工具。
- **产品利率报价**：货币、期限、Fixed/Floating 矩阵，应使用 pricing 工具。

## 交易能力

交易查询和汇总工具用于生成管理人维度日报、空头汇总、CSV 导出和 North Info 导出。查询时应明确日期、客户、管理人和汇总口径。

## 数据归属

客户、交易、报价属于业务插件数据，不应写入核心平台主库。
