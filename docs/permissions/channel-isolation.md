# 渠道隔离

渠道是权限判断的一部分。CLI、飞书、企微、Telegram 即使使用同一个用户身份，也不会被视为完全相同的执行环境。

## CLI

CLI 是系统管理入口。只有在 `channel=cli` 且当前用户满足系统管理员条件时，才可以执行系统级管理操作。

## Bot 渠道

飞书、企微和 Telegram 通过 Bot 入口进入 Agent。Bot 渠道不自动获得系统管理员能力，权限由 Agent 绑定、成员关系和工具列表共同决定。

## 设计效果

- 防止 Bot 消息触发系统级配置变更。
- 支持同一平台服务多个 Agent 和多个 Bot app。
- 让 `/status`、skill、tool 等展示结果以当前 Agent 上下文为准。
