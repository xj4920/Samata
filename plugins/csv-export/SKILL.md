---
name: CSV 文件导出
description: 将 JSON 数据数组导出为 CSV 文件
---

## 何时使用
- 用户要求将数据导出为 CSV 文件时
- 已有结构化数据（JSON 对象数组）需要转换为 CSV 格式时

## 使用步骤
1. 准备好 JSON 对象数组形式的数据
2. 调用 export_csv，传入数据数组和文件名
3. 可通过 columns 参数指定输出列及顺序

## 注意事项
- data 参数为 JSON 对象数组，每个对象代表一行
- 不指定 columns 时，自动取第一行对象的所有 key 作为表头
- 自动处理 CSV 转义（逗号、引号、换行符）
